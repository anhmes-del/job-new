# CAREEROS PRO – AI Career Operating System

## One‑Click Launch
```bash
git clone https://github.com/your-repo/careeros-pro.git
cd careeros-pro
cp backend/.env.example backend/.env
# edit backend/.env with your API keys
make install
```

## Modules
CV Diagnostic, ATS Optimizer, Achievement Rewriter, LinkedIn Branding, Cover Letter, Interview Copilot, Job Tracker

AI Skill Gap, Salary Intelligence, Career Path, Market Demand

Recruiter portal, matching engine, communication (email/SMS), document engine, knowledge RAG

Monetization (Stripe), admin panel, analytics
